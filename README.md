# 🚀 Laravel Blog Management System

A full-stack Blog Management System built as part of a technical assessment assignment. The application features a clean, reader-first minimalist frontend design, interactive AJAX/jQuery filtering mechanics, and a robust administrative panel with an integrated Rich Text Editor (TinyMCE).

## 🛠️ Technology Stack & Tools Used
*   **Backend Framework:** PHP 8.2+ / Laravel 11 Engine
*   **Database Management:** Local Development via MySQL (XAMPP) | Cloud Production via MySQL (InfinityFree)
*   **Frontend Interfaces:** Blade Template Engine + Modern Minimalist Responsive Grid Styling
*   **Asynchronous JavaScript:** Native jQuery Core Library using AJAX (`$.get`) API endpoints
*   **Rich Text Formatter:** TinyMCE Content Editing Framework CDN Integration
*   **DevOps Deployment Control:** Google Antigravity Agent-First IDE Orchestration System + InfinityFree Cloud Hosting

## ✨ Implemented Core Assignment Features
1.  **Dynamic Public Blog Feed:** Renders detailed content cards (Title, Cover Graphics, Excerpt, Date Badges) pulled from active cloud database collections.
2.  **Zero-Refresh AJAX Filtering Engine:** Captures selection dropdown adjustments (Categories, Timeline Dates) and runs optimized, debounced AJAX queries to modify matching blog collections dynamically without reloading the parent webpage.
3.  **Comprehensive Admin CRUD Panel:** Fully isolated backend administrative ecosystem equipped to securely Create, Read, Update, and Delete database records with seamless persistent handling.
